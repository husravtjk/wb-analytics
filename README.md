# WB Финансовая Аналитика

Веб-дашборд для анализа финансовых отчётов Wildberries.

## Возможности

- **Дашборд** — KPI-карточки, графики продаж по неделям, тренд выплат, P&L, доля брендов
- **Товары** — таблица товаров с редактируемой себестоимостью и расчётом чистой прибыли
- **Загрузка отчётов** — импорт еженедельного отчёта WB и отчёта по товарам (xlsx)
- **Налоги** — настраиваемые ставки УСН и НДС
- Тёмная тема, адаптивный дизайн

## Стек

- Express + Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Drizzle ORM + SQLite
- Recharts (графики)
- xlsx (парсинг Excel-файлов)

## Запуск

```bash
npm install
npm run dev
```

Приложение запустится на http://localhost:5000

## Продакшн

```bash
npm run build
NODE_ENV=production node dist/index.cjs
```

## Использование

1. Откройте раздел «Загрузить»
2. Загрузите еженедельный отчёт WB (.xlsx)
3. Загрузите отчёт по товарам (.xlsx)
4. Настройте ставки налогов (УСН 6%, НДС 5%)
5. Перейдите в «Товары» — заполните себестоимость
6. На «Дашборде» увидите полную финансовую картину
